# Dashboard Charts Rules

Chart library: [LayerChart](https://next.layerchart.com/)

---

## GENERAL CARD STYLES

All cards share the same base styles:

| Element | Token |
|---|---|
| Padding | `--spacing-4` |
| Border radius | `--rounded-6` |
| Shadow | `--shadow-sm` |
| Card title | `--card-main-title` |
| Main number | `--card-number-md` |
| Text after number | `--card-text` |
| Emoji | `--h3` |
| Chart axis labels | `--help-text` |

## CARD 1 — BILANCIO CALORICO

### Header
- **Title:** "Bilancio calorico" + emoji 🍽️ (always visible, top right)
- **Main number:** "[X] /[total] giorni vicino al target"
  - X = days within target range in the selected period
  - Total = total days in the selected period (changes with filter)

### Chart — Weekly view
- Bar chart with a center line and dots
- X axis: day of week letter + day number (e.g. "S 10", "D 11")
- Y axis: 5 lines, always integers (no fractions). If value is fractional, round for axis labels and show exact value in tooltip only.

### Chart — Monthly / 3-month view
- Bar chart with a center line, no dots on the line
- X axis: dates (e.g. "18/12", "26/12")
- Y axis: same rule as weekly (integers on axis, exact in tooltip)

### Colors

| Element | Token |
|---|---|
| Center line (target) | `--primary-action` |
| Poco sopra (bars) | `--primary-surface` |
| Poco sotto (bars) | `--ciano-200` |
| Troppo sopra (bars) | `--danger-surface` |
| Troppo sotto (bars) | `--danger-200` |

### Logic — bar color rules
- Bar is **Poco sopra** if calories are slightly above target
- Bar is **Poco sotto** if calories are slightly below target
- Bar is **Troppo sopra** if calories are significantly above target → use danger color
- Bar is **Troppo sotto** if calories are significantly below target → use danger color

## CARD 2 — OBIETTIVO DI PESO

### Header
- **Title:** "Obiettivo di peso → [goal]kg"
- **Main number:** "[X] kg persi → da [starting]kg a [current]kg attuale"
  - Starting = weight at the beginning of this goal
  - Current = most recent logged weight

### Chart — Weekly view
- Line chart with a dot for each day
- X axis: day of week letter + day number (e.g. "S 10", "D 11")
- Y axis: 4 lines, always integers. If value is fractional, round for axis labels and show exact value in tooltip only.

### Chart — Monthly / 3-month view
- Line chart without dots
- X axis: dates
- Y axis: same rule as weekly

### Colors

| Element | Token |
|---|---|
| Weight goal line (dotted) | `--neutral-surface-light` |
| Daily weight line (solid) | `--primary-action` |

## CARD 3 — CALORIE ATTIVE

### Header
- **Title:** "Calorie Attive" + emoji 🔥 (always visible, top right)
- **Main number:** "[X] kcal giorno (media [period])"
  - X = sum of all active kcal in the period ÷ number of days in the period
  - Period label changes with filter: "media settimana" / "media mese" / "media 3 mesi"

### Chart — All views
- Bar chart, one bar per day
- Weekly: 7 bars, X axis shows day of week letter + day number
- Monthly / 3-month: X axis shows dates

### Y axis
- Number of lines: between 3 and 5, based on the variation in user data
  - If variation is small → 3 lines
  - If variation is large → 5 lines
- Always integers. If value is fractional, round for axis labels and show exact value in tooltip only.

### Colors

| Element | Token |
|---|---|
| Bars | `--nutrient-attive-surface` |
